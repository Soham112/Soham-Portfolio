# Soham Patil Portfolio

Personal portfolio built with Next.js 14, Tailwind CSS, and Framer Motion. 

Features a warm, minimalist aesthetic mapping Soham's journey as a Data Scientist and Machine Learning Engineer.

## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React + Standard SVGs
- **Deployment**: Vercel ready

## Getting Started

First, install the dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
